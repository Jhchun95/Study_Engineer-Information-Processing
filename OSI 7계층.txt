OSI 7계층

* 통신이 일어나는 과정을 7단계로 정의한 국제 통신 표준 규약
* 각 계층은 독립되어있다 → 용이성, 유지관리의 수월성

1. 물리계층 : 데이터를 전기적인 신호로 변환해서 주고받는 기능을 진행하는 공간. 장비로는 통신 케이블, 허브가 존재한다.
		 리피터,허브
			
2. 데이터링크 계층 : 물리계층으로 송/수신되는 정보 확인하고 오류 없는 통신을 위해 여러 역할을 수행한다.
		 MAC 주소를 통신하고, 장비로는 브릿지와 스위치가 존재한다.
		 브리지, 스위치
3. 네트워크 계층 : 데이터를 목적지까지 가정 안전하고 빠르게 전달하는 기능. 라우터를 통해 경로를 선택하여
	           IP주소를 지정하고 경로에 따라 패킷을 전달해준다. 장비로는 라우터가 존재한다.
		   라우터
4. 전송 계층 : 두 호스트 시스템으로부터 발생하는 데이터의 흐름을 제공한다.
		
5. 세션 계층 : 통신 시스템 사용자 간의 연결을 유지 및 설정한다.
6. 표현 계층 : 세션 계층 간의 주고받는 인터페이스를 일관성 있게 제공한다.
	   프로토콜 : 
7. 응용 계층 : 사용자가 네트워크에 접근할 수 있도록 서비스를 제공한다.
